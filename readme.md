Установка:

```bash
1. Клонировать репозиторий
git clone https://github.com/andrey3304/The_greatest_project_of_our_time.git

2. Установить зависимости
pip install -r requirements.txt

3. Настройка окружения
cp .env.example .env

4. Запуск
python app.py
```
